# Wall-Asterisk 1.0.1
Wall **** is a local chat.

# Environment
- Linux
- Following header:
  - ncurses.h
  - pthread.h

# Installation
1. change directory to 'wa100/myllib' :\
    `wa100$ cd mylib`
2. make file :\
    `wa100/mylib$ make`
3. change directory to 'wa100/wa' :\
    `wa100/mylib$ cd ../wa`
4. make file :\
    `wa100/wa$ make`

# Install headers
Ex: Ubuntu 20.04 LTS

- ncurses.h:\
`$ sudo apt install libncurses5-dev`

- pthread.h\
`$ sudo apt install libpthread-dev`
